1. 처음에 빔프로젝트로 웹사이트 화면 띄우면 회원가입을 해야함
2. 빔프로젝트 화면에 QR을 핸드폰으로 체크하면 핸드폰이 회원가입 페이지로 이동
3. 회원가입 완료하면 프로필 생성 페이지로 이동
4. 프로필 생성후 선호 재료, 선호 카테고리, 알러지 정보 설문 페이지 이동
5. 4번 까지 이동하면 빔 프로젝트 화면 으로 돌아와서 프로필에 등록한 얼굴 이미지로 안면인식으로 접속
6. 이후 레시피 추천 페이지로 모션이용해서 이동
7. 레시피 추천 페이지에서 위로 스와이트 모션하면 즐겨찾기 페이지 이동, 아래로 모션은 전체 레시피 보여주는 사이트 이동
8. 전체 레시피 보여주는 사이트에서 레시피 선택 모션하면 레시피 상세 가이드 페이지로 이동
